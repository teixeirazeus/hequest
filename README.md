Simple and hackable request for dart.

## Installing

1.  Add dependencies to `pubspec.yaml`

    ```yaml
    dependencies:
        cronos:
            git:
                url: https://github.com/teixeirazeus/hequest
    ```

2.  Run pub get.

    ```shell
    flutter pub get
    ```

3.  Import package.

    ```dart
    import 'package:hequest/hequest.dart';
    ```
